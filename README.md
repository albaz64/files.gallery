## 说明

程序破解原理是修改 JS 的验证函数, **check.php** 作用是替换官网验证服务器的返回内容.(写check代码: 两分钟; 找到正确的返回值: 超过一天)

程序配置文件默认路径是 `_files\config\config.php` (本仓库的默认隐藏两个破解文件), 其他参数具体用法参照官网: https://www.files.gallery/docs/config/
